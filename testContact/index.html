Ülesande kirjeldus

Leidke endale tiim 2-4 liiget, võib ka üksi teha, kui kedagi ei leia.

Tehke sellest repositooriumist fork ja laadige töö GitHub'i üles.

Võite leida erinevaid alternatiive, et kuidas koodi koos kirjutada, aga üks võimalus on installida VS Code'ile laiendus Live Share.

Mis osutus raskeks

Kontakti lehel osutus raskeks google maps api lisamine, kuna algselt tahtis Kristo selle API kujul panna, niiet raskusi ei pakkunud selle lisamine, vaid pigem google maps platform-i õppimine. Lõpuks otsustasime seda mitte teha ja lihtsalt iframe-ga panna.
